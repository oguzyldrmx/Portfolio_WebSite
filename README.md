## Getting Started

For Backend Sanity:
* go to the back-end folder in console
```sh
  sanity start
  ```
* for manage sanity
```sh
  sanity manage
  ```
For Frontend:

*npm
```sh
  npm install @sanity/client @sanity/image-url framer-motion node-sass react-icons
  ```
  ```sh
  npm start
  ```
